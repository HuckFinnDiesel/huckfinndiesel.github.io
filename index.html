Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://huckfinndiesel.github.io/repo/ (Give it the name "Magicality").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "Firefly", and install repository.firefly-0.0.1.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "Firefly Repo"
  • Go into the "Video add-ons" section in the repo, and you'll find Firefly

Enjoy!
